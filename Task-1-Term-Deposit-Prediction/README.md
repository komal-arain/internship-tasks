# Term Deposit Subscription Prediction

## Objective
Predict whether a bank customer will subscribe to a term deposit.

## Approach
- Data preprocessing and encoding
- Logistic Regression & Random Forest models
- Evaluation using Confusion Matrix, F1 Score, ROC Curve
- Model explainability using SHAP

## Results
Random Forest performed better with higher F1-score.
SHAP analysis identified duration and previous campaign outcome as important features.

## Business Insight
Target customers with longer call duration and positive previous outcomes.
